# Prompt para Geração de Componentes

Use este prompt quando quiser que o Copilot gere componentes específicos:

```text
Crie o componente [NOME_DO_COMPONENTE] para Power Apps Canvas App.

Requisitos:
- Reutilizável em múltiplas telas.
- Propriedades de entrada: [liste as propriedades].
- Propriedades de saída: [liste as saídas/eventos].
- Estilo consistente com o design system (cores, espaçamento, tipografia).
- Suporte a estados: default, hover, disabled, loading, error.
- Acessível: contraste adequado, foco visível, labels claras.
- Responsivo: adapta-se a diferentes larguras de tela.

Entregue:
1. Estrutura de controles do Power Apps.
2. Fórmulas de propriedades principais.
3. CSS se usar HTML Text.
4. Guia de uso com exemplos.
5. Checklist de acessibilidade aplicado.
```

## Exemplos de uso
- "Crie o componente CardMetric para exibir KPIs com título, valor, variação e ícone."
- "Crie o componente FormGroup com label, input, helper text e validação visual."
- "Crie o componente EmptyState com ícone, título, descrição e botão de ação."
